# experimental-vibes
Some AI, vibe-coding and agentic experimentation

This is an experiment using Claude Code to:

(1) Scrape military range activity - such as live firing or low-flying - from GOV.UK

(2) compile this into a suitable data object (.JSON)

(3) create a GOV.UK style service that helps people check if there is any military activity planned in their area today.

This explores a prototype service that could conceivably be useful to dog walkers, horse riders, military planners...

Notes:

This code does a one-time scrape of the data and then generates a standalone hmml page (/docs/index.html)

Hence the data is a snapshot, it is not live

This is a personal learning project and not associated with GOV.UK, UK government or UK forces.
